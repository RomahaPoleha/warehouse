# Warehouse — Система учёта оборудования

Django-приложение для управления складом электронных компонентов и оборудования. Позволяет отслеживать остатки, оформлять заявки на выдачу и формировать отчёты.


---

##  Функционал

###  Для обычного пользователя:
- Просмотр каталога оборудования по категориям:
  - ИК-рамки
  - Фронтальные платы и коннекторы
  - Платы на базе Android / Windows
  - T-CON платы
  - Кабели
  - Оборудование для видеостен
- Поиск по названию, модели и модификации
- Оформление заявки на выдачу оборудования
- Просмотр истории своих заявок

### Для сотрудника (администратора):
- Просмотр всех заявок в статусе «Ожидает»
- Подтверждение или отклонение заявок
- Автоматическое списание остатков при подтверждении
- Экспорт всего каталога оборудования в Excel с форматированием
- История обработанных заявок

### Безопасность и надёжность:
- Авторизация пользователей (Django Auth)
- Разграничение прав: `@login_required`, `@staff_member_required`
- Атомарные транзакции при списании (`@transaction.atomic`)
- Валидация: нельзя запросить больше, чем есть на складе

## Установка и запуск (локально)

### 1. Клонирование репозитория
```bash
git clone https://github.com/RomahaPoleha/warehouse.git
cd warehouse
```


### 2. Создание виртуального окружения
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### 3.  Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4.  Применение миграций
```bash
python manage.py migrate
```

### 5.  Запуск сервера
```bash
python manage.py runserver
```
